Spring Cloud Stream Applications can be used with [Spring Cloud Data Flow](https://cloud.spring.io/spring-cloud-dataflow/) to create, deploy, and orchestrate message-driven microservice applications. {% endcapture %} {% capture main_content %} Spring Cloud Stream Application Starters are standalone executable applications that communicate over messaging middleware such as Apache Kafka and RabbitMQ. These applications can run independently on variety of runtime platforms including: Cloud Foundry, Apache Yarn, Apache Mesos, Kubernetes, Docker, or even on your laptop. ## Features * Run standalone as Spring Boot applications * Compose microservice as streaming pipelines in Spring Cloud Data Flow * Consume microservice applications as maven or docker artifacts * Override configuration parameters via command-line, environment variables, or YAML file * Provide infrastructure to test the applications in isolation * Download as starters from this version of [Spring Initializr](https://start-scs.cfapps.io/) ## Available Applications | __Source__ | __Processor__ | __Sink__ | | ------- | --------- | ---- | ---- | | file | aggregator | aggregate-counter | | ftp | tcp | | | | throughput | | | | websocket | ## Quick Start Step 1 - Download the latest _time-source_ application from [here](https://repo.spring.io/libs-release/org/springframework/cloud/stream/app/time-source-kafka/) [eg: /2.1.0.RELEASE/time-source-kafka-2.1.0.RELEASE.jar] Step 2 - Download the latest _log-sink_ application from [here](https://repo.spring.io/libs-release/org/springframework/cloud/stream/app/log-sink-kafka) [eg: /2.1.0.RELEASE/log-sink-kafka-2.1.0.RELEASE.jar] Step 3 - Start Kafka Step 4 - Run **Time Source** and Bind to *ticktock* Topic ``` java -jar time-source-kafka-***.jar --spring.cloud.stream.bindings.output.destination=ticktock ``` Step 5 - Run **Log Sink** and Bind to *ticktock* Topic ``` java -jar log-sink-kafka-***.jar --spring.cloud.stream.bindings.input.destination=ticktock ``` Step 6 - Verify *ticktock* ouput in `log-sink` console ## Stream App Starters and Spring Cloud
